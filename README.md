# password-generator
To create a password generator first create a list of all numbers , create a list of all alphabetes , create a list of all symbols.
Create variable to ask user needs for example numbers_need , letters_need , symbols_need. Input statments.
Create a placeholder list and assign a variable to it.
Using for loop add all the items which will fit to user requirements into that list placeholder by using random module.
Use a method of random module which will shuffle the the list items.
Use for loop to add all the characters of passwordlist.
Print the password using f-string so that it look good.
This is the method to create a password generator.
